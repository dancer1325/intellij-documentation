# Change Class Signature dialog

Use the Change Class Signature dialog to perform the [Change Class Signature](change-signature.html#change_class_signature_refactoring) refactoring.

Manage the formal type parameters using the available controls (see the table that follows). You can add and remove parameters as well as change their order.

Click Refactor to perform the refactoring right away. Click Preview to see the potential changes prior to actually performing the refactoring. (These will be shown in the [Find tool window](find-tool-window.html).)

Icon| Shortcut| Description  
---|---|---  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.add.svg)| `Alt+Insert`| Use this icon or shortcut to add a parameter. Specify the parameter name and default type in the Name and the Default Value fields respectively.  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.remove.svg)| `Alt+Delete`| Use this icon or shortcut to remove the selected parameter.  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.up.svg)| `Alt+Up`| Use this icon or shortcut to move the selected parameter one line up in the list.  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.down.svg)| `Alt+Down`| Use this icon or shortcut to move the selected parameter one line down in the list.
