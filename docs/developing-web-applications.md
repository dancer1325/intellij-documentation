## Developing Web applications

  1. [Enable Web application development support](enabling-web-application-support.html) to get the necessary libraries, the basic folder hierarchy, and the application descriptor.

  2. [Populate the Web module](web-application-web-module-structure.html): create the required Java classes, configure servlets, filters, listeners, and references.

  3. [Configure the required static Web content resources](web-application-static-content.html).

  4. [Configure Web application deployment](configure-web-app-deployment.html).

  5. Deploy and run the application.
