# EJB

Support for developing container-based business applications using [EJB](https://projects.eclipse.org/projects/ee4j.ejb) is not bundled with IntelliJ IDEA. You can install the [Jakarta EE: Enterprise Java Beans (EJB)](https://plugins.jetbrains.com/plugin/16939-jakarta-ee-enterprise-java-beans-ejb-/) plugin from JetBrains Marketplace as described in [Install plugins](managing-plugins.html).

You can find the documentation for EJB in [earlier versions of IntelliJ IDEA Help](https://www.jetbrains.com/help/idea/2020.1/ejb.html).
