## Log settings

Depending on the run/debug configuration type, you can access logs by clicking the Modify options button or opening the Logs tab. If there is neither, it means that the selected run/debug configuration type doesn't support logs.

![The 'Modify options' button in the 'Run/Debug Configurations' dialog](https://resources.jetbrains.com/help/img/idea/2025.3/logs-modify-options.png)

![The 'Logs' tab in the 'Run/Debug Configurations' dialog](https://resources.jetbrains.com/help/img/idea/2025.3/logs-tab.png)

![No 'Modify options' button or 'Logs' tab in the 'Run/Debug Configurations' dialog](https://resources.jetbrains.com/help/img/idea/2025.3/no-logs.png)
