# Languages and Frameworks

When you select the Languages & Frameworks category in the left-hand pane, its main subcategories are listed in the right-hand part of the dialog.

![Settings: Languages and Frameworks](https://resources.jetbrains.com/help/img/idea/2025.3/settings-languages.png)

If the necessary language or framework is not on the list, you can add its support by installing a plugin. For more information about installing plugins, refer to [Install plugins](managing-plugins.html).
