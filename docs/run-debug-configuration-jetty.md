# Run/Debug Configuration: Jetty Server

Support for [Eclipse Jetty](https://www.eclipse.org/jetty/) is not bundled with IntelliJ IDEA. You can install the [Jetty plugin](https://plugins.jetbrains.com/plugin/18721-jetty) from JetBrains Marketplace as described in [Install plugins](managing-plugins.html).

You can find the documentation for Jetty Server in [earlier versions of IntelliJ IDEA Help](https://www.jetbrains.com/help/idea/2021.3/run-debug-configuration-jetty.html).
