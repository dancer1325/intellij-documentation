## Create an HTML file

  1. Go to File | New, and then select HTML File from the list. 

  2. In the dialog that opens, type the name of the new file without any extension. You can type the whole directory structure before the new filename. If the nested directories do not yet exist, they will be created: 

![Create an HTML file](https://resources.jetbrains.com/help/img/idea/2025.3/ws_create_html_file.png)

IntelliJ IDEA creates a stub file based on the [HTML file template](using-file-and-code-templates.html) and opens it in the editor.
