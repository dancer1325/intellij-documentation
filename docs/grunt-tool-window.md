## Before you start

  1. Download and install [Node.js](http://nodejs.org/#download). 

  2. Install the grunt-cli package (Grunt command-line interface) globally and install the grunt package as a development dependency. For more information, refer to [Installing Grunt](using-grunt-task-runner.html#installingGrunt).
