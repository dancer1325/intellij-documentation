# Smart Keys settings: Scala

Use this settings page to configure typing assistance features in Scala.

For more information about Scala support in IntelliJ IDEA, see [Get started with Scala](get-started-with-scala.html).

Item| Description  
---|---  
Insert pair quotes for multiline string| With this option selected, IntelliJ IDEA inserts closing triple quotes when you try to add a multiline string. ![Scala triple quotes](https://resources.jetbrains.com/help/img/idea/2025.3/scala_quotes.png)  
Wrap single expression body with closing brace after '{'| With this option selected, IntelliJ IDEA [wraps](scala-features-overview-scala.html#wrap_unwrap_scala_methods) the body of an expression with braces.  
Upgrade simple string into interpolated after ${| With this option selected, IntelliJ IDEA recognizes the interpolated string and adds a closing brace when you insert ${ inside a simple string. ![Scala interpolated string](https://resources.jetbrains.com/help/img/idea/2025.3/scala_simple_string_to_interpolated.png)
