# Cloud Foundry

Support for building and deploying applications in the clouds with [Cloud Foundry](https://www.cloudfoundry.org/) is not bundled with IntelliJ IDEA.

The [Cloud Foundry plugin](https://plugins.jetbrains.com/plugin/13854-cloud-foundry) is no longer supported. The latest IDE version compatible with the plugin is IntelliJ IDEA 2023.1.

You can find the documentation about the Cloud Foundry support in [earlier versions of IntelliJ IDEA Help](https://www.jetbrains.com/help/idea/2023.1/working-with-clouds.html).
