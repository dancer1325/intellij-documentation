# Package Search

Support for searching, evaluating, and managing dependencies with Package Search is not bundled with IntelliJ IDEA

The [Package Search](https://plugins.jetbrains.com/plugin/12507-package-search) plugin has been deprecated.

You can find the Package Search documentation in [earlier versions of IntelliJ IDEA Help](https://www.jetbrains.com/help/idea/2024.2/package-search.html).
