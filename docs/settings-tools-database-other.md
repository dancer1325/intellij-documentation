## Modify Object

Item| Description  
---|---  
Confirm cancellation for dialogs that modify schema| Show a confirmation dialog when you close Create or Modify dialogs. Close means clicking Cancel or pressing `Escape`. If the checkbox is cleared, the dialog closes instantly.For more information about the dialogs, refer to [Create and Modify dialogs](create-and-modify-dialogs.html).
