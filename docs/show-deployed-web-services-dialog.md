# Show Deployed Web Services dialog

Tools | Web Services | Show Deployed Web Services

Use this dialog to have a list of all Web services deployed at the current server host displayed.

Item| Description  
---|---  
Context name| In this field, type the [application context](http://download.oracle.com/javaee/1.4/tutorial/doc/WebApp3.html) under which you want to see the deployed Web services.  
Status| This read-only field shows information on the status of the target server and hints in the format of the data specified in the Context name field.
