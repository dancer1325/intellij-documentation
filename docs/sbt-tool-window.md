## sbt

Item| Description  
---|---  
![Show option menu](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.settings.svg)| Click this button to open the menu for configuring the current view and changing the tool window viewing modes. The menu items are options that you can turn on or off. An active item contains a check mark on the left side of its name.The available options in the menu are as follows:

  * Group Modules: select this option if you want to group modules in the project.
  * Group Tasks: select this option if you want to group your tasks. For example, build tasks will be included into the build directory.
  * Show Ignored: select this option to display the ignored projects.
  * Show Toolbar: select this option to show the toolbar for your sbt projects.
  * [Project tool window](project-tool-window.html#viewing_modes) Pinned Mode, Docked Mode, Floating Mode, Split Mode â€“ these options let you control general appearance and behavior of the tool window, refer to [Tool Windows Viewing Modes](viewing-modes.html)
  * Group Tabs: deselect this option to see the views on separate tabs if more than one view is available in a tool window.
  * Move to: select this option to move the sbt tool window to either top, left or right.
  * Resize: select this option to resize the sbt tool window.

  
![Hide tool window](https://resources.jetbrains.com/help/img/idea/2025.3/app.general.hideToolWindow.svg)| Hide the tool window `Shift+Escape`.To hide all the tool windows, press `Ctrl+Shift+F12`.
