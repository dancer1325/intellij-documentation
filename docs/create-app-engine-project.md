## Installing Go SDK

Ensure that the provided path to the folder with Go SDK includes bin and src folders.

### Select a local copy of the Go SDK

  1. Press `Ctrl+Alt+S` to open settings and then select Languages & Frameworks | Go | GOROOT.

  2. Click the Add SDK button (![the Add SDK button](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.add.svg)) and select Local.

  3. In the file browser, navigate to the SDK version that is on your hard drive.

  4. Click Open.

![Installing Go SDK](https://resources.jetbrains.com/help/img/idea/2025.3/go_ij_installing_go_sdk.png)



### Download the Go SDK

  1. Press `Ctrl+Alt+S` to open settings and then select Languages & Frameworks | Go | GOROOT.

  2. Click the Add SDK button (![the Add SDK button](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.add.svg)) and select Download.

  3. From the Version list, select the SDK version.

  4. In the Location field, specify the path for the SDK. To use a file browser, click the Browse icon (![the Browse icon](https://resources.jetbrains.com/help/img/idea/2025.3/app.actions.menu-open_dark.svg)).

  5. Click OK.

![Installing Go SDK](https://resources.jetbrains.com/help/img/idea/2025.3/go_ij_downloading_go_sdk.png)
