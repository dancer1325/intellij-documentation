## Enabling Minimalistic Mode

### Via JPA Buddy Tool Window

To enable minimalistic mode, click ![Minimalistic Mode](https://resources.jetbrains.com/help/img/idea/2025.3/Minimalistic_Mode.svg) in the JPA Buddy tool window.

![tool-window-enable](https://resources.jetbrains.com/help/img/idea/2025.3/tool-window-enable.png)

### Via Settings

Another way to activate minimalistic mode is via settings. Open Settings | Tools | JPA Buddy | Designer Settings and switch View mode to Minimalistic.

![settings-minimalistic](https://resources.jetbrains.com/help/img/idea/2025.3/settings-minimalistic.png)

Read more about Designer Settings [here](jpa-buddy-entity-designer.html#designer-settings).
