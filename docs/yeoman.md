# Yeoman

Support for developing applications based on the [Yeoman framework](https://yeoman.io/) is not bundled with IntelliJ IDEA. You can install the [Yeoman](https://plugins.jetbrains.com/plugin/7987-yeoman) plugin from JetBrains Marketplace as described in [Install plugins](managing-plugins.html).

You can find the documentation for Yeoman in [earlier versions of IntelliJ IDEA Help](https://www.jetbrains.com/help/idea/2022.2/yeoman.html).
