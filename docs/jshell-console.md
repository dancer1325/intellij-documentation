# JShell Console

JShell Console is not bundled with IntelliJ IDEA. You can install the [JShell](https://plugins.jetbrains.com/plugin/28663-jshell) plugin from JetBrains Marketplace as described in [Install plugins](managing-plugins.html).

You can find the documentation for JShell Console in IntelliJ IDEA in [earlier versions of IntelliJ IDEA Help](https://www.jetbrains.com/help/idea/2025.2/jshell-console.html).
