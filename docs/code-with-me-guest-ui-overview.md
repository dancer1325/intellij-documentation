## Settings

You can configure options that are related to appearance, code editing, formatting, and so on. You can also install plugins that are related to code editing, formatting, schemes, and themes.

### Configure Appearance and Behavior

  1. Press `Ctrl+Alt+S` to open settings and then select Appearance and Behavior.

  2. Using various options in different sections, you can change themes, accessibility options, UI options, notification popups, and so on.

![the Appearance and Behavior settings](https://resources.jetbrains.com/help/img/idea/2025.3/cwm_appearance_behavior.png)

For more information about the settings options, refer to [Appearance & Behavior](appearance-and-behavior.html).




### Install plugin

  1. Press `Ctrl+Alt+S` to open settings and then select Plugins.

  2. From the options on the right, select the Marketplace tab and select a plugin you want to install.

You can select the Installed tab and check a list of installed plugins.

![the Plugins settings](https://resources.jetbrains.com/help/img/idea/2025.3/installed_plugins.png)

For more information about the settings, refer to [Plugins](plugins-settings.html).

Even if you have IntelliJ IDEA installed, you will still see only those plugins that are available for JetBrains Client.




### Configure editor options

  1. Press `Ctrl+Alt+S` to open settings and then select Editor.

  2. From the available options, you can configure font, colors of user markers in a session, console colors, and so on.

![the Editor settings](https://resources.jetbrains.com/help/img/idea/2025.3/cwm_editor_settings.png)

For more information, refer to [Editor](settings-editor.html).




As a guest, you are able to change the user name for a session only from the JetBrains Client welcome screen.

The welcome screen appears when you close the JetBrains Client window while in the session.

![JetBrains Client welcome screen](https://resources.jetbrains.com/help/img/idea/2025.3/JB_Client_welcome_screen.png)

As a host, you can configure a username for a session through the [Code With Me settings](#tools).

### Configure a username for the session

  1. Press `Ctrl+Alt+S` to open settings and then select Tools.

  2. From the options on the left, select Code With Me.

  3. For the options on the right, in the User name field, enter a name you want to use for the session and click OK to save the changes. Alternatively, you can click Use System Name to use it in the session.

![the Code With Me settings](https://resources.jetbrains.com/help/img/idea/2025.3/cwm_settings_user_name.png)

To apply the new name for all new sessions, you need to restart JetBrains Client.




For more information about options in the debugger and in the advanced settings, refer to [Debugger settings](settings-debugger.html) and [Advanced Settings](advanced-settings.html). Note that these settings options in JetBrains Client are limited to appearance, behavior, code editing, and formatting.
