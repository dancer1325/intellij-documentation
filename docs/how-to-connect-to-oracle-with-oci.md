## Required files and settings

  * Running Oracle instance with access permissions for your user.

  * Configured tnsnames.ora file. For full information about tnsnames.ora file, refer to the [official documentation](https://docs.oracle.com/cd/E11882_01/network.112/e10835/tnsnames.htm).
