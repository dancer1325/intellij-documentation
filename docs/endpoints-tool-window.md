## Supported frameworks

IntelliJ IDEA recognizes endpoints from the following frameworks:

  * Jakarta EE WebSocket

  * Java EE WebSocket

  * JAX-WS

  * JAX-RS

  * JAX-RS Client

  * gRPC/Protobuf

  * MicroProfile Rest Client

  * Spring Feign Client

  * Spring MVC

  * Spring Reactive WebSocket

  * Spring RestTemplate

  * Spring WS

  * Spring WebClient

  * Spring Web Services

  * Micronaut HTTP Controllers

  * Micronaut HTTP Client

  * Micronaut Management Endpoints

  * Micronaut WebSocket

  * Ktor

  * Helidon MP

  * Helidon SE

  * Quarkus

  * OpenAPI 3

  * Swagger 2

  * WSDL

  * Retrofit 2

  * OkHttp 3+
