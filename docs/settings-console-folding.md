## Folding patterns

Use the following buttons to manage the folding patterns from the two lists:

![The Add button](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.add.svg) Add `Alt+Insert`
    

Open the Folding Pattern dialog and type a new pattern.

![The Edit button](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.edit.svg) Remove `Alt+Insert`
    

Remove the selected pattern from the list.

![The Edit button](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.edit.svg) Edit `Enter`
    

Open the Folding Pattern dialog and edit the selected pattern.
