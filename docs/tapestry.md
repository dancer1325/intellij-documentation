# Tapestry

Support for developing Web applications based on the [Tapestry framework](http://tapestry.apache.org) is not bundled with IntelliJ IDEA.

The [Tapestry plugin](https://plugins.jetbrains.com/plugin/14589-tapestry) is [no longer supported](https://blog.jetbrains.com/idea/2022/12/end-of-support-for-the-tapestry-jboss-seam-and-vaadin-plugins/). The latest IDE version compatible with the plugin is IntelliJ IDEA 2022.3.

You can find the documentation for the Tapestry support in IntelliJ IDEA in [earlier versions of IntelliJ IDEA Help](https://www.jetbrains.com/help/idea/2020.2/discover-intellij-idea.html). The plugin source code is available on [GitHub](https://github.com/JetBrains/intellij-obsolete-plugins/tree/master/tapestry).
