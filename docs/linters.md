## View problems and apply quick-fixes in the editor

  1. In the editor, hover over the highlighted problem. IntelliJ IDEA shows a tooltip with a description of the problem.

![JavaScript linters: errors and warnings are highlighted, the description of a problem is shown in a tooltip.](https://resources.jetbrains.com/help/img/idea/2025.3/ws_js_linters_common_show_description.png)

To resolve the problem, click <Linter name>: Fix '<rule name>' or press `Alt+Shift+Enter`.

  2. To resolve all the detected problems in the current file, click More actions (`Alt+Enter`) and select <Linter name>: Fix current file from the list.

![JavaScript linters: resolving problems](https://resources.jetbrains.com/help/img/idea/2025.3/ws_js_linters_common_show_description_intention_actions.png)

Alternatively, press `Alt+Enter` and choose the relevant suggestion from the list.
