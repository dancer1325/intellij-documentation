## Uninstall using the Toolbox App

If you installed IntelliJ IDEA [using the Toolbox App](installation-guide.html#toolbox), do the following:

  * Open the Toolbox App, click the settings icon ![Tool actions](https://resources.jetbrains.com/help/img/idea/2025.3/app.actions.more.svg) for the necessary instance, and select Uninstall.
