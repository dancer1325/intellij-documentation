# Select Branch

The popup opens when you click Browse ![the Browse button](https://resources.jetbrains.com/help/img/idea/2025.3/app.general.ellipsis.svg) or press `Shift+Enter` to select the path to the target branch.

Use this dialog to select the relevant branch or working copy.

Item| Description  
---|---  
Trunk| Choose this option to set the current trunk as the target branch or working copy.  
Branches| Choose this option to select the relevant branch in the Branches list.  
Tags| Choose this option to select the relevant tag in the Tags list.  
Configure Branches| Choose this option to open the [Configure Subversion Branches](configure-subversion-branches.html) dialog and compose a list of branches you work with.
