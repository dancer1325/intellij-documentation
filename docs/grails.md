# Grails

Support for the [Grails](https://grails.org/) framework is not bundled with IntelliJ IDEA. You can install the [Grails plugin](https://plugins.jetbrains.com/plugin/18504-grails) from JetBrains Marketplace as described in [Install plugins](managing-plugins.html).

The latest compatible version of IntelliJ IDEA is 2021.3.

You can find the documentation for Grails support in [earlier versions of IntelliJ IDEA Help](https://www.jetbrains.com/help/idea/2021.1/grails.html).
