# Create new constructor

Item| Description  
---|---  
Visibility| Select one of the options to change visibility scope of the constructor.  
Name| Specify the parameter name.  
Parameters  
Type| Specify the parameter type.Note that [code completion](auto-completing-code.html) works here. As you type, the suggestion list shrinks to show the matching options. Non-existent type is marked as a syntax error.  
Add/Remove| Use these buttons to add/remove parameters.  
Move Up/Down| Use these buttons to reorder parameters by moving them up or down in the parameter list.  
Exceptions  
Add/Remove| Use these buttons to add/remove exceptions. For each new exception, specify its type.Note that [code completion](auto-completing-code.html) works here. As you type, the suggestion list shrinks to show the matching options. Non-existent type is marked as a syntax error.  
Move Up/Down| Use these buttons to reorder exceptions by moving them up or down in the list.  
Signature Preview| This read-only area shows the draft constructor signature as you fill in the dialog fields.
