## Path Variables page

For more information about configuring variables, refer to [Path variables](absolute-path-variables.html).

Item| Description  
---|---  
Reset| Revert path variables and the list of ignored variables to their initially saved states.  
Name| This field shows the name of a path variable (readonly).  
Value| This field shows the value of a path variable (readonly).  
![the Add button](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.add.svg) or `Alt+Insert`| Create a new path variable.  
![the Edit button](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.edit.svg) or `Enter`| Edit the selected path variable.  
![the Remove button](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.remove.svg) or `Alt+Delete`| Delete the selected path variable.  
Ignored Variables| List the names of the variables that should be ignored. Use semicolons `;` to separate the list items.
