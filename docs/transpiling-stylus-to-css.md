## Before you start

  1. Make sure you have [Node.js](http://nodejs.org/) on your computer. 

  2. Install the Stylus and File Watchers plugins on the Settings | Plugins page, tab Marketplace, as described in [Installing plugins from JetBrains Marketplace](managing-plugins.html#install_plugin_from_repo).
