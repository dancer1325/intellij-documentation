## Toolbar

Item| Tooltip and shortcut| Description  
---|---|---  
![View Options](https://resources.jetbrains.com/help/img/idea/2025.3/app-client.expui.general.inspections.inspectionsEye.svg)| View Options| Filter out code issues by their severity and configure their sorting. When no grouping or sorting options are selected, the issues are listed in the order they appear in the file.   
![the Show Quick Fixes button](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.codeInsight.intentionBulb.png)| Show Quick-Fixes `Alt+Enter`| Show quick-fixes available for the selected problem.   
![the Open Editor Preview button](https://resources.jetbrains.com/help/img/idea/2025.3/app-client.expui.general.previewVertically.svg)| Open Editor Preview| Open the preview pane to view the selected issue in its source context right in the Problems window. Note that this preview is a normal editor where you can change the code and apply available quick-fixes.
