# Vaadin

Support for developing applications based on the [Vaadin framework](https://vaadin.com/) is not bundled with IntelliJ IDEA.

The [Vaadin Designer](https://plugins.jetbrains.com/plugin/9519-vaadin-designer) plugin is [no longer supported](https://blog.jetbrains.com/idea/2022/12/end-of-support-for-the-tapestry-jboss-seam-and-vaadin-plugins/). The latest IDE version compatible with the plugin is IntelliJ IDEA 2022.3.

You can find the documentation for the Vaadin support in IntelliJ IDEA in [earlier versions of IntelliJ IDEA Help](https://www.jetbrains.com/help/idea/2019.2/getting-started-with-vaadin.html). The plugin source code is available on [GitHub](https://github.com/JetBrains/intellij-obsolete-plugins/tree/master/vaadin-framework)
