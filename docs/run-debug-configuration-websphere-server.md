# Run/Debug Configuration: WebSphere Server

Support for [WebSphere Server](https://www.ibm.com/cloud/websphere-application-server) is not bundled with IntelliJ IDEA. You can install the [WebSphere Server plugin](https://plugins.jetbrains.com/plugin/18723-websphere) from JetBrains Marketplace as described in [Install plugins](managing-plugins.html).

You can find the documentation for WebSphere Server in [earlier versions of IntelliJ IDEA Help](https://www.jetbrains.com/help/idea/2021.3/run-debug-configuration-websphere-server.html).
