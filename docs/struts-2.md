# Struts 2

Support for developing applications based on the [Struts 2 framework](https://struts.apache.org/) is not bundled with IntelliJ IDEA.

The [Struts 2 plugin](https://plugins.jetbrains.com/plugin/1698-struts-2) is no longer supported. The latest IDE version compatible with the plugin is IntelliJ IDEA 2023.1.

You can find the documentation about Struts 2 support in [earlier versions of IntelliJ IDEA Help](https://www.jetbrains.com/help/idea/2020.1/struts-2.html). The plugin source code is available on [GitHub](https://github.com/JetBrains/intellij-obsolete-plugins/tree/master/struts2).
