## Prerequisites

  1. Install and enable Vagrant plugin as described in [Installing plugins from JetBrains Marketplace](managing-plugins.html#install_plugin_from_repo).

  2. Install [Vagrant](http://www.vagrantup.com/) and [Oracle's VirtualBox](https://www.virtualbox.org/) applications.

  3. Make sure virtualization is enabled on your computer.
