## Disable double-key shortcuts

Do the following to disable the default double-key shortcuts, Double `Shift` for [Search everywhere](searching-everywhere.html) and Double `Ctrl` for [Run Anything](running-anything.html).

  1. Press `Ctrl+Alt+S` to open settings and then select Advanced Settings.

  2. Scroll to the User Interface section and set the Disable double modifier key shortcuts checkbox: 

![Disabling the double-key shortcuts](https://resources.jetbrains.com/help/img/idea/2025.3/cl_doublekey_shourtcuts_disable.png)
