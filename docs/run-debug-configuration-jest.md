## Before you start

  1. Download and install [Node.js](http://nodejs.org/#download). 

  2. Make sure the JavaScript and TypeScript plugin is enabled in the settings. Press `Ctrl+Alt+S` to open settings and then select Plugins. Click the Installed tab. In the search field, type JavaScript and TypeScript. For more information about plugins, refer to [Managing plugins](managing-plugins.html).
