## Create a new Play project

  1. Launch the [New Project wizard](new-project-wizard.html). If no project is currently opened in IntelliJ IDEA, click New Project on the welcome screen. Otherwise, select File | New | Project from the main menu.

  2. From the options on the left, under the Generators node, select Play. 

![New Project dialog](https://resources.jetbrains.com/help/img/idea/2025.3/play_create.png)
  3. Follow the steps suggested in the wizard and click Create. 

IntelliJ IDEA generates the Play project with the appropriate structure, dependencies, and an executable application.

![Play project editor and tool windows](https://resources.jetbrains.com/help/img/idea/2025.3/ready_play_project.png)
