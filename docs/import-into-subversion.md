## Toolbar buttons

Item| Tooltip and Shortcut| Description  
---|---|---  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.add.svg)| Add Repository Location| Click this button to configure a new [repository location](configuring-subversion-repository-location.html).  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.edit.svg)| Edit Location URL| Click this button to edit the URL address of the selected repository.  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.remove.svg)| Discard Location URL| Click this button to discard the selected repository location and remove it from the list.  
![showHideDetails.png](https://resources.jetbrains.com/help/img/idea/2025.3/showHideDetails.png)| Show/Hide Details| Click this button to display the details for each node below the repository location (changelist number, user name, date and time of the last change).  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.actions.refresh.svg)| Refresh `Ctrl+F5`| Click this button to refresh the view.  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.collapseAll.svg)| Collapse All `Ctrl+NumPad -`| Click this button to have all the nodes below all the repository locations collapsed.
