# Configuring Subversion Branches

IntelliJ IDEA allows you to compose a list of parent folders of the branches you work with. This list will be displayed every time you perform any operation with branches, for example, when you synchronize your local working copy, compare branches, etc. 

Branches are configured in the [Configure Subversion Branches](configure-subversion-branches.html) dialog.

To configure Subversion branches, do the following:

  1. Open the [Configure Subversion Branches dialog](configure-subversion-branches.html): in the Version Control tool window `Alt+9`, switch to the [Subversion Working Copies Information](subversion-working-copies-information-tab.html) tab, and then click the Configure Branches link.

  2. In the Trunk location field, specify the URL of your repository trunk. Type the address or click the Browse button ![the Browse button](https://resources.jetbrains.com/help/img/idea/2025.3/app.general.ellipsis.svg) and select the trunk location in the Select Repository Location dialog that opens. This dialog shows a tree of all branches and tags under the [repository root](configuring-subversion-repository-location.html).

  3. In the Branch locations area, make up a list of folders where the branches you need in your work are stored. Use the ![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.add.svg) and ![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.remove.svg) buttons to add/remove branches to/from the list.
