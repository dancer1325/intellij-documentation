# Jakarta Server Faces (JSF)

Support for [Jakarta Server Faces](https://javadoc.io/doc/jakarta.faces/jakarta.faces-api) (JSF), formerly known as JavaServer Faces, is not bundled with IntelliJ IDEA. You can install the [Jakarta EE: Server Faces (JSF)](https://plugins.jetbrains.com/plugin/18583-jakarta-ee-server-faces-jsf-) plugin from JetBrains Marketplace as described in [Install plugins](managing-plugins.html).

You can find the documentation for JSF in [earlier versions of IntelliJ IDEA Help](https://www.jetbrains.com/help/idea/2021.3/javaserver-faces-jsf.html).
