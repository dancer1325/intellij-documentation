## Before you start

  1. Make sure you have [Node.js](http://nodejs.org/) on your computer. 

  2. Make sure the JavaScript and TypeScript and JavaScript Debugger required plugins are enabled on the Settings | Plugins page, tab Installed. For more information, refer to [Managing plugins](managing-plugins.html). 

  3. Install the React and Vite plugins on the Settings | Plugins page, tab Marketplace, as described in [Installing plugins from JetBrains Marketplace](managing-plugins.html#install_plugin_from_repo).
