# Charts Settings

Use this page to customize the appearance of charts in your notebooks. Select the contrast or default theme. Click ![edit the theme](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.edit.svg) to modify the theme colors or click ![clone the theme](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.copy.svg) to clone the theme and customize it later.

In the lower part of the page, you can use a sample chart to preview how your color changes will look on different types of charts.
