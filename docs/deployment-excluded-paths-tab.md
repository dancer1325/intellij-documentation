# Deployment: Excluded Paths Tab

Use this tab to configure a list of local and remote folders that you do not want to sync. If you want to exclude files and folders using patterns (for example, all files with a specific extension), use the Exclude items by name field on the [Options](settings-deployment-options.html) page of the deployment settings section.

![Add item](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.add.svg) Add `Alt+Insert`| Add an exclusion to the list and select the type of the path from the popup menu:

  * Local path: the absolute path to a folder on the local system that you don't want to be synced over to the remote.
  * Deployment path: the absolute path to the folder on the remote system that you don't want to be synced to the local system.

Click ![the Browse button](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.open.svg) and choose the path to the required folder in the dialog that opens.  
---|---  
![Remove item](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.remove.svg) Remove `Alt+Delete`| Remove the selected exclusion from the list.
