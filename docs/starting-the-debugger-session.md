## Debug multiple tasks

  * To run or debug multiple tasks simultaneously, open the run widget menu in the toolbar and select the corresponding run/debug configurations while holding down the `Ctrl` key.

![%alt](https://resources.jetbrains.com/help/img/idea/2025.3/debug-multiple-rc.png)



For launching multiple tasks using a single run/debug configuration, refer to [Run compound tasks](run-debug-multiple.html).
