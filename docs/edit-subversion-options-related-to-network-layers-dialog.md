## Toolbar Options

Item| Tooltip and shortcut| Description  
---|---|---  
![the Add button](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.add.svg)| Add `Alt+Insert`| Click this button to have a new configuration profile added to the list.  
![the Remove button](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.remove.svg)| Delete `Delete`| Click this button to remove the selected profile from the list.  
![the Copy button](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.copy.svg)| Copy `Ctrl+D`| Click this button to have a copy of the selected profile created.
