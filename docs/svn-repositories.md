## Toolbar

Item| Tooltip and shortcut| Description  
---|---|---  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.add.svg)| Add Repository Location| Click this button to configure a new [repository location](configuring-subversion-repository-location.html). The New Repository Location dialog opens, where you can select a repository URL in the drop-down list that contains previously added URL addresses.  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.edit.svg)| Edit Location Url| Click this button to edit the URL of the selected repository.  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.remove.svg)| Discard Location| Click this button to remove the selected repository from the list.  
![showHideDetails.png](https://resources.jetbrains.com/help/img/idea/2025.3/showHideDetails.png)| Show/Hide Details| Click this button to display the details for each node under the repository location (changelist number, user name, date and time of the last change).  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.actions.refresh.svg)| Refresh`Ctrl+F5`| Click this button to refresh the view.  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.expandAll.svg) ![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.collapseAll.svg)| Expand All/Collapse All`Ctrl+NumPad +`| Click this button to expand/collapse all nodes.  
![the Help icon](https://resources.jetbrains.com/help/img/idea/2025.3/app.actions.help.svg)| Help `F1`| Open a browser and show the corresponding help page.  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.closeSmall.svg)| Close| Click this button to close the tool window.
