# Find Usages. Variable Options

This section describes Field, Variable, or Parameter usage search options in the Find Usages dialog.

The dialog opens when you click ![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.settings.svg) in the Show Usages popup which lists all the occurrences of the symbol at caret.

Item| Description  
---|---  
Skip results tab with one usage| Select this checkbox to be navigated directly to the found usage without the Find tool window displayed when only one usage is found.  
Scope| In this area, specify the [scope](configuring-scopes-and-file-colors.html) of search. Select a pre-defined scope from the list or click the Browse button ![the Browse button](https://resources.jetbrains.com/help/img/idea/2025.3/app.general.ellipsis.svg) to open the [Scopes](settings-scopes.html) dialog, where you can define a custom scope.  
Open results in new tab| Select this checkbox have the results of each search shown in a separate tab of the Find Results window. If the checkbox is cleared, the search results will overwrite the contents of the current tab.
