## Productivity tips

### Use the quick switcher

  1. Press `Ctrl+`` or select View | Quick Switch Scheme from the main menu.

  2. In the Switch popup, select Theme, and then select the desired interface theme.




### Create shortcuts

You can map the Theme action to your [preferred key combination](configuring-keyboard-and-mouse-shortcuts.html).
