## Before you start

  1. Install and enable the Bower plugin on the Settings | Plugins page, tab Marketplace, as described in [Installing plugins from JetBrains Marketplace](managing-plugins.html#install_plugin_from_repo). 

  2. Download and install [Node.js](http://nodejs.org/#download).
