# Configure HTTP proxy in Subversion

Subversion stores the http proxy settings in the servers file in the user's runtime configuration area (~/.subversion/auth/ on Unix systems or <USER HOME>/Application Data/Subversion/auth/ on Windows).

You can configure the Subversion proxy settings in two ways:

  * [Edit the servers file](http://svnbook.red-bean.com/en/1.1/svn-book.html#svn-ch-7-sect-1) manually.

  * Configure proxy settings directly from IntelliJ IDEA. Do the following:

    1. Open the Version Control | Subversion | Network settings page `Ctrl+Alt+S`.

    2. Click the Edit Network Options button and specify the proxy settings in the [Edit Subversion Options Related to Network Layers](edit-subversion-options-related-to-network-layers-dialog.html) dialog that opens.
