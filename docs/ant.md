# Getting started with Ant

The [Ant](https://ant.apache.org/) plugin is not bundled with IntelliJ IDEA. You can install [Ant](https://plugins.jetbrains.com/plugin/23025-ant) from JetBrains Marketplace as described in [Install plugins](managing-plugins.html).

The latest compatible version of IntelliJ IDEA is 2023.2.

You can find the Ant documentation in [earlier versions of IntelliJ IDEA Help](https://www.jetbrains.com/help/idea/2023.2/ant.html).
