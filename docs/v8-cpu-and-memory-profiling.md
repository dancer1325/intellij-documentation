## Before you start

  * Make sure the Node.js plugin is enabled in the settings. Press `Ctrl+Alt+S` to open settings and then select Plugins. Click the Installed tab. In the search field, type Node.js. For more information about plugins, refer to [Managing plugins](managing-plugins.html).
