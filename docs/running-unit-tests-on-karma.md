## Before you start

  1. Download and install [Node.js](http://nodejs.org/#download). 

  2. Make sure the JavaScript and TypeScript and Karma required plugins are enabled on the Settings | Plugins page, tab Installed. For more information, refer to [Managing plugins](managing-plugins.html).
