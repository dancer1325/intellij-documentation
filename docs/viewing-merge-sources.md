## To view the merge source

  1. Open the Version Control tool window, and switch to the [Repository](version-control-tool-window-repository-and-incoming-tabs.html) tab.

  2. Select the required changelist, and then choose a file from this changelist in the Changed Files pane.

  3. Select Show History from the context menu, or click ![](https://resources.jetbrains.com/help/img/idea/2025.3/app.vcs.history.svg) on the toolbar. The [History tab](version-control-tool-window-history-tab.html) for the selected file will be displayed. For each revision, the Merge sources field shows the revisions that were used as sources for the merge. 

![mergeSourcesHistory.png](https://resources.jetbrains.com/help/img/idea/2025.3/mergeSourcesHistory.png)
  4. To view all paths affected by a merge, select the relevant row and press `Alt+Shift+A` or click ![](https://resources.jetbrains.com/help/img/idea/2025.3/app.actions.listChanges.svg) on the toolbar. 

Review affected paths in the dialog that opens:

![pathsAffected.png](https://resources.jetbrains.com/help/img/idea/2025.3/pathsAffected.png)
