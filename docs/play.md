# Play Framework

Support for developing Web applications based on the [Play framework](https://www.playframework.com/) is not bundled with IntelliJ IDEA.

The [Play Framework plugin](https://plugins.jetbrains.com/plugin/14583-play-framework) is no longer supported. The latest IDE version compatible with the plugin is IntelliJ IDEA 2023.1.

You can find the documentation for the Play framework support in IntelliJ IDEA in [earlier versions of IntelliJ IDEA Help](https://www.jetbrains.com/help/idea/2020.1/play.html). The plugin source code is available on [GitHub](https://github.com/JetBrains/intellij-obsolete-plugins/tree/master/play).
