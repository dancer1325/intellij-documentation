## Move tool windows

  * Click and drag the tool window icon or the tool window header. 

The option to move tool windows by dragging the header can be [disabled in settings](advanced-settings.html#drag_toolwindow_title_bar).

  * Right-click the toolbar icon of the tool window, navigate to Move to, and select where to attach the tool window.

  * Alternatively, you can click the tool window options menu ![Options](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.moreVertical.svg) or right-click the [tool window header](tool-windows.html#general-tool-windows-layout) and select where to attach the tool window under Move to.

![Tool window options menu: Move to](https://resources.jetbrains.com/help/img/idea/2025.3/ij_tool_window_moveto.png)



You can save the current location of the tool windows and their custom sizes as a [layout](tool-window-layouts.html).

To restore the default layout, in the main menu, go to Window | Layouts | Default or press `Shift+F12`.
