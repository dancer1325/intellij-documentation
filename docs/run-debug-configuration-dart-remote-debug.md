## Dart Remote Debug-specific configuration settings

Item| Description  
---|---  
Search Sources in| In this field, specify the Dart project to debug if your IntelliJ IDEA project contains [several Dart projects](dart.html#working_with_multiple_dart_packages_in_project) configured as content roots. Select the Dart project from the list or click ![the Browse button](https://resources.jetbrains.com/help/img/idea/2025.3/app.general.ellipsis.svg) and select the required folder in the dialog that opens. See [Adding a Dart project (package) to an IntelliJ IDEA project](dart.html#working_with_multiple_dart_packages_in_project).
