# Integrate to Branch

Use this dialog to specify the options for integrating changes into a branch.

Item| Tooltip| Description  
---|---|---  
Source branch URL| This read-only field shows the URL address of the source branch.  
Target branch URL| This read-only field shows the URL address of the target branch.  
Integrate into working copy| From this list, select the path to the local working copy into which the changes will be integrated.  
Ignore whitespaces| Select this option if whitespaces are not important.  
Try merge, but make no changes| Select this option to preview merge results by enabling the `--dry-run` switch of the `svn` command. If this option is unchecked, sources are merged silently.  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.add.svg)| Add| Click this button to add a working copy to the list.  
![](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.remove.svg)| Remove| Click this button to remove the selected working copy from the list.
