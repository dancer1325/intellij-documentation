## Paths to resources outside the project

You can use [path variables](absolute-path-variables.html) to point to resources outside the project. Add configured path variables the same way as you add macros â€“ with dollar signs before and after the variable: `$MY_VAR$`.

### Create a new path variable

  1. Press `Ctrl+Alt+S` to open settings and then select Appearance & Behavior | Path Variables. 

![Path Variables dialog](https://resources.jetbrains.com/help/img/idea/2025.3/path-variables-dialog.png)
  2. Click ![the Add button](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.add.svg), enter the name and value of the variable, and click OK to apply the changes.
