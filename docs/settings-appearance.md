## Accessibility

Option| Description  
---|---  
Zoom| Zoom in and out the entire IDE, increasing or decreasing the size of all UI elements at once.You can also access this setting from the main menu: select View | Appearance | Zoom IDE.  
Use custom font| Specify custom font family and size to use in the IntelliJ IDEA UI.  
Support screen readers| Enable screen reader support for IntelliJ IDEA. For more information, refer to [Accessibility](accessibility.html).  
Use contrast scrollbars| Make the editor [scrollbar](accessibility.html#scrollbars_contrast) more visible.  
Adjust colors for red-green vision deficiency| Adjust UI colors to better perceive colors in case of protanopia and deuteranopia.
