## Before you start

  1. Download and install [Node.js](http://nodejs.org/#download). 

  2. Configure a Node.js runtime in your project as described in [Configuring a local Node.js interpreter](developing-node-js-applications.html#ws_node_configure_local_node_interpreter), in [Using Node.js on Windows Subsystem for Linux](developing-node-js-applications.html#ws_node_wsl), or in [Configuring remote Node.js runtimes](configuring-remote-node-interpreters.html).
