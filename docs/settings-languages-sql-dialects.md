## Supported SQL dialects

IntelliJ IDEA supports the following SQL and NoSQL dialects:

  * Amazon DynamoDB

  * Amazon Redshift

  * Apache Cassandra

  * Apache Derby

  * Apache Hive

  * Apache Spark

  * Azure SQL Database

  * ClickHouse

  * CockroachDB

  * Couchbase Query Service

  * Exasol

  * Google BigQuery

  * Generic SQL

  * Greenplum

  * H2

  * HSQLDB

  * IBM Db2 iSeries

  * IBM Db2 LUW

  * IBM Db2 z/OS

  * MariaDB

  * Microsoft SQL Server

  * MongoDB

  * MySQL

  * Oracle

  * Oracle SQL*Plus

  * PostgreSQL

  * Redis

  * Snowflake

  * SQLite

  * SQL2016

  * Sybase ASE

  * Vertica
