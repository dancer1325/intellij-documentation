## Dart web-specific configuration settings

Item| Description  
---|---  
HTML file| In this field, specify the path to the HTML file that references the Dart code to run or debug. Most likely it is <dart project root>/web/index.html  
Webdev port| This field shows the port on which the [webdev server](https://dart.dev/tools/webdev#serve) will start the Dart web app, refer to [Running and debugging Dart web applications](dart.html#debug_dart_web_app_webdev_daemon).
