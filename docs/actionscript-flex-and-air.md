# Flash

Support for developing applications based on ActionScript and Flex is not bundled with IntelliJ IDEA. You can install the [Flash/Flex](https://plugins.jetbrains.com/plugin/14508-flash-flex) plugin from JetBrains Marketplace as described in [Install plugins](managing-plugins.html).

You can find the documentation about ActionScript and Flex support in [earlier versions of IntelliJ IDEA Help](https://www.jetbrains.com/help/idea/2020.1/actionscript-flex-and-air.html).
