# Main version control shortcuts

Use the VCS Operations Popup (`Alt+``) or <your_VCS> | VCS Operations Popup in the main menu to quickly invoke any VCS-related commands.

The list of actions in the popup depends on the currently enabled VCS.

No VCS Enabled| VCS Enabled  
---|---  
![A list of available actions when a VCS is disabled](https://resources.jetbrains.com/help/img/idea/2025.3/vcs_disabled.png)| ![A list of available actions when a VCS is enabled](https://resources.jetbrains.com/help/img/idea/2025.3/vcs_enabled.png)
