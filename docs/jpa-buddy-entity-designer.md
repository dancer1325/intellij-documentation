## Persistence tool window

Most of the features previously available in JPA Structure are now available in the [Persistence tool window](persistence-tool-window.html). Additionally, JPA Buddy adds the following features to the Persistence tool window:

  * Create JPA Entity, Generate JPA Entity from a Java or Kotlin class

  * Create Spring Data JPA Repository and Spring Data Projection

  * Create DTO based on JPA entities

  * View DTOs and projections for each entity


![Persistence tool window](https://resources.jetbrains.com/help/img/idea/2025.3/persistence_tool_window_with_jpa_buddy.png)

You can set up JPA Structure visibility in [visual designer settings](#designer-settings).
