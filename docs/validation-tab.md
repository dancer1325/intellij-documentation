# Validation tab

File | Project Structure | Artifacts | <artifact> | Validation

Use this tab to define server-specific validators that perform real-time validation of applications.

It is available only for artifacts that can be deployed to a server, and not available for artifacts of the types Jar and Other.

Application Server-specific validators| The list of current application server validation extensions to apply.  
---|---  
![the Add button](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.add.svg) Add| Select the class that implements the relevant validator.  
![the Remove button](https://resources.jetbrains.com/help/img/idea/2025.3/app.expui.general.remove.svg) Remove| Remove the selected entry from the list.
