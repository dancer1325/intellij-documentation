## View and manage diagrams

### Pan and zoom

  * To pan and zoom the diagram, use your touchpad, scroll bars or the mouse wheel for vertical scrolling and `Shift` \+ mouse wheel for horizontal scrolling. You can also use `Shift` \+ mouse wheel for zooming.

  * To use the magnifier tool, keep the `Alt` key pressed and hover over the desired areas of the diagram. Use the mouse wheel to extra zoom the diagram area within the magnifier tool. 

![Magnifier tool](https://resources.jetbrains.com/help/img/idea/2025.3/db_magn_tool.png)



### Enable column comments

  1. Generate a diagram. For more information about generating a diagram, refer to [Generate a diagram for a database object](#db_diagrams).

  2. Click the Comments button (![the Comments button](https://resources.jetbrains.com/help/img/idea/2025.3/app.general.balloon.svg)).

![Enable column comments](https://resources.jetbrains.com/help/img/idea/2025.3/db_enable_column_comments.png)



### Apply colors to diagram objects

  1. In the Database tool window, right-click an object and select Tools | Set Color.

  2. Select a color and click OK. Refresh the diagram to apply the color by clicking the Refresh button ().

![Display of table colors](https://resources.jetbrains.com/help/img/idea/2025.3/display_of_table_colors.png)



### Find diagram elements

To locate your element on the diagram, do the following:

  1. Press `Ctrl+F` to open the Diagram Elements popup window.

  2. Start typing the name of the element, select it, and press `Enter`.


![Find a diagram element](https://resources.jetbrains.com/help/img/idea/2025.3/db_diagrams_find_element.png)
