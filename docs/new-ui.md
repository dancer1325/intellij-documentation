## Themes, icons, fonts

  * The new Light, Dark, and Light with Light Header [color themes](user-interface-themes.html) have improved contrast and a consistent color palette.

![Light theme in the new UI](https://resources.jetbrains.com/help/img/idea/2025.3/new_ui_light_theme.png)

![Light theme with light header in the new UI](https://resources.jetbrains.com/help/img/idea/2025.3/ij_new_ui_light_header_theme.png)

![Dark theme in the new UI](https://resources.jetbrains.com/help/img/idea/2025.3/new_ui_dark_theme.png)

  * The Inter font is used for the UI on all supported OSs.

  * A new icon set has more distinguishable shapes and colors designed for legibility and visual balance.
