# Run/Debug Configuration: WebLogic Server

Support for [WebLogic](https://www.oracle.com/ru/java/weblogic/) is not bundled with IntelliJ IDEA. You can install the [WebLogic plugin](https://plugins.jetbrains.com/plugin/18722-weblogic) from JetBrains Marketplace as described in [Install plugins](managing-plugins.html).

You can find the documentation for WebLogic in [earlier versions of IntelliJ IDEA Help](https://www.jetbrains.com/help/idea/run-debug-configuration-websphere-server.html).
